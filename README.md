# mintCV3
Naprogramujte celulárny paralelný genetický algoritmus (CPGA) pre hľadanie globálneho minima Eggholder funkcie 10 premenných . Architektúru CPGA si zvoľte (prednášky). Do spoločného obrázku porovnajte priemer 5 spustení CPGA, ostrovného PGA a jednoduchého (jednopopulačného) GA (SGA), všetky s rovnakým počtom jedincov, ako je jedincov v celom PGA resp. CPGA.
